# 버티는 투자 프로젝트

## 프로젝트 개요
- 사이트명: 버티는 투자
- 슬로건: 흔들리면서 배우는 가치투자 실전 일지
- URL: value.newtechtree.com (Cloudflare Pages 배포)
- 저장소: jkgoods/value-diary

## 콘셉트
전문가가 아닌 개인이 마법공식/가치투자를 실전으로 따라가는 날것의 기록 공개. 투자 권유 아님. 분기 1회 종목 선정 기록, 수시로 심리 메모.

## 투자 방식
- 마이스노우볼 시총 필터 → 부채비율/ROC → 네이버증시 PER 3~10 확인 → 당기순이익 확인
- 분기 1회 5종목 매수, 종목당 10만원, 1년 보유 후 매도
- 마법공식 기반 가치투자

## 디자인 방향
- 라이트 미니멀, 흰 배경
- 최대 너비 680px 중앙 정렬
- AI 대시보드 느낌 배제, 사람 냄새 나는 블로그 느낌
- 폰트: -apple-system, Apple SD Gothic Neo 계열

## 페이지 구성
- 메인(index.html): 히어로 + 최근 매수 5종목 + 투자일지 목록
- 포트폴리오 페이지: 전체 종목 테이블 (최대 20종목)
- 투자일지 페이지: 글 목록
- 소개 페이지: 면책문구 포함

## 메인 페이지 구조
1. 네비: 버티는 투자 로고 / 포트폴리오·투자일지·소개 메뉴
2. 히어로: "흔들리면서 배우는 가치투자 실전 일지" + 설명 + 메타정보
3. 최근 매수 종목 5개 + 전체보기 링크 + 요약(총투자/평가/수익률/종목수)
4. 투자일지 목록 (날짜·태그·제목·본문 미리보기)
5. 푸터: 면책문구

## 일지 태그 종류
- 심리기록: 보유 중 감정 메모
- 스크리닝: 분기 종목 선정 과정
- 결과기록: 분기 마감 수익률
- 소개: 사이트 소개

## 기술 스택
- 순수 HTML/CSS (프레임워크 없음)
- Cloudflare Pages 정적 배포
- GitHub 저장소 push시 자동 배포
- 현재가 자동 갱신: 추후 pykrx 스크립트로 구현 예정 (Mac Mini cron)

## 법적 고지
모든 페이지 하단에 고정:
"본 기록은 개인 투자 일지이며 투자 권유가 아닙니다. 모든 투자 결정과 그 결과에 대한 책임은 본인에게 있습니다."
